## Логика упаковки

Профили должны выглядеть как одна взрослая специализация:

- `CRM/API + интеграции`
- `React/frontend + кабинеты`
- `доработка сайтов + e-commerce`
- `bugfix/support/stabilization`

Второй слой:

- `Telegram`
- `Python/backend`
- `автоматизация`

Не используйте в заголовке и первом экране слово `парсеры` как главный маркер.
