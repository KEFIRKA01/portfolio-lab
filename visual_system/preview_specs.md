# Preview Specs

## Shot list для каждого проекта

Минимальный комплект после деплоя:

1. Главный экран или основная dashboard-сцена.
2. Главный рабочий сценарий.
3. Внутренний экран администратора или оператора.
4. Настройки / интеграции / журнал событий.
5. Мобильный или адаптивный экран, если применимо.
6. Архитектурная схема.

## GIF plan

Для каждого проекта нужен 1 короткий ролик `15-30 сек`:

- пользователь делает действие;
- система меняет состояние;
- админ/менеджер видит результат;
- на финальном кадре остаётся ценность проекта.

## Единые подписи к обложкам

- `Demo project`
- `Case-ready`
- `GitHub + live demo`
- `Architecture included`
