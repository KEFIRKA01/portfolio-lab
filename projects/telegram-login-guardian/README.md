# Защитник входов в Telegram

`Защитник входов в Telegram` — большой витринный Telegram-бот.

## Фокус

- Нужно подтверждать входы через Telegram, отсекать подозрительные сессии и давать безопасный путь для доверенных устройств.
- Показывает Telegram-решение с акцентом на безопасность: MFA-логика, риск-сигналы и расследования.
- Направление: `подтверждение входа, доверие к устройству, усиленная проверка`

## Крупные блоки функционала

- подтверждение входа
- риск-скоринг устройства
- усиленная проверка
- журнал сессий
- отзыв активной сессии

## Ключевые сценарии

- Новый девайс
- Подозрительный IP
- Повышенный риск
- Доверенный вход
- Ревизия активных сессий

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `безопасный маршрут входа, подтверждение сессий, MFA, антифрод`;
- хорошо продаёт нетипичные запросы заказчиков.
