# Бот ресторанной выручки

`Бот ресторанной выручки` — большой витринный Telegram-бот.

## Фокус

- Нужно принимать брони через Telegram, держать очередь ожидания, контролировать депозиты и давать хостес удобный поток подтверждений.
- Показывает сервисный и доходный бот с сегментацией гостей и управлением загрузкой.
- Направление: `ресторан, брони, очередь ожидания, депозиты, допродажи`

## Крупные блоки функционала

- бронь и очередь ожидания
- депозиты
- гостевые сегменты
- апсейлы
- стол и зал

## Ключевые сценарии

- Новая бронь
- День рождения
- Корпоратив
- Очередь ожидания
- Подтверждение депозита

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `брони, очередь ожидания, депозиты, оценка гостей`;
- хорошо продаёт нетипичные запросы заказчиков.
