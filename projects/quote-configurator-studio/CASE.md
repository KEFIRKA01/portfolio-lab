# Кейс: Quote Configurator Studio

## Задача

Показать второй сильный frontend-угол: не просто страница для заявки, а инструмент сборки предложения и квалификации объёма до разговора с менеджером.

## Решение

Собран демонстрационный configurator studio, где пользователь выбирает тип продукта, модули, сложность и получает ориентировочную смету вместе с payload для менеджера.
