# Центр продления подписок

`Центр продления подписок` — большой витринный Telegram-бот.

## Фокус

- Нужно вовремя продлевать подписки, удерживать клиентов, корректно управлять статусами доступа и мягко работать с риском оттока.
- Раскрывает направление регулярной выручки и жизненного цикла клиента: подписки, удержание, специальные предложения и статусы доступа.
- Направление: `подписки, продления, мягкое дожатие оплаты, предложения лояльности`

## Крупные блоки функционала

- маршрут продления
- работа с неоплаченным счётом
- персональные предложения
- статусы доступа
- сводка удержания

## Ключевые сценарии

- Продление
- Неудачный платеж
- Пауза подписки
- VIP предложение
- Риск оттока

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `подписки, повторная работа с оплатой, удержание, контроль доступа`;
- хорошо продаёт нетипичные запросы заказчиков.
