# Ассистент первичного юридического приёма

`Ассистент первичного юридического приёма` — большой витринный Telegram-бот.

## Фокус

- Нужно быстро принимать юридические обращения, собирать базовые документы, фильтровать срочность и направлять кейсы подходящему юристу.
- Показывает экспертный сценарий первичного приёма с аккуратной логикой вопросов и маршрутизацией.
- Направление: `юридический приём, первичная сортировка обращения, чеклист документов`

## Крупные блоки функционала

- юридический первичный приём
- маршруты по практике
- чеклисты
- срочность
- дашборд координатора

## Ключевые сценарии

- Первичное обращение
- Срочный спор
- Интеллектуальная собственность
- Проверка договора
- Сбор документов

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `юридический приём, первичная сортировка, чеклисты`;
- хорошо продаёт нетипичные запросы заказчиков.
