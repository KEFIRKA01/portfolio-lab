# Координатор складских отгрузок

`Координатор складских отгрузок` — большой витринный Telegram-бот.

## Фокус

- Нужно согласовывать окна забора товара, собирать документы перевозчиков и разгружать складскую команду от ручных согласований.
- Добавляет направление supply-chain и показывает умение строить операционные маршруты с окнами и ограничениями.
- Направление: `склад, окна забора, рампы, документы перевозчика`

## Крупные блоки функционала

- окна и рампы
- документы перевозчика
- резерв остатков
- приоритезация
- операторская сводка

## Ключевые сценарии

- Новое окно
- Рампа занята
- Документы перевозчика
- VIP отгрузка
- Проблема с остатком

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `складские операции, окна забора, назначение рампы`;
- хорошо продаёт нетипичные запросы заказчиков.
