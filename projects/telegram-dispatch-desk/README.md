# Telegram-диспетчер выездных задач

Демонстрационный Telegram-сценарий работы для распределения выездных задач между командами и контроля ETA.

## Что показывает проект

- диспетчеризацию и выездные операции;
- crew assignment;
- ETA breach;
- handoff между сменами;
- escalation по невзятым задачам.
