# Бот онбординга партнёров

`Бот онбординга партнёров` — большой витринный Telegram-бот.

## Фокус

- Нужно сопровождать новых партнёров в едином Telegram-потоке: документы, статусы, запуск и сигналы блокировок.
- Расширяет профиль в сторону B2B-онбординга, готовности к запуску и координации между аккаунт-менеджером, юридическим и операционным блоком.
- Направление: `онбординг партнёров, документы, комплаенс, чеклист запуска`

## Крупные блоки функционала

- онбординг стадий
- документы
- готовность к запуску
- комплаенс
- статусы запуска

## Ключевые сценарии

- Новый партнер
- Документы не готовы
- Интеграция заблокирована
- Чеклист запуска
- Корпоративный запуск

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `онбординг партнеров, поток документов, чеклист запуска`;
- хорошо продаёт нетипичные запросы заказчиков.
