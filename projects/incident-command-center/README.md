# Центр управления инцидентами

Демонстрационный центр корреляции инцидентов, сервисных сигналов и сценариев реагирования.

## Что показывает проект

- инцидент-центр вместо одиночных проверок доступности;
- корреляцию сигналов в один инцидент;
- оценку критичности;
- ответственность и следующие шаги;
- нестандартные аварийные сценарии.
