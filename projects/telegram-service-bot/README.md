# Telegram-бот сервисных заявок

Демонстрационный Telegram-бот для приёма заявок, менеджерской обработки и внутреннего рабочего сценария.

## Что показывает проект

- сценарий сбора заявки;
- автоматическую маршрутизацию по статусам;
- уведомления менеджеру;
- операторскую логику поверх Telegram;
- сводку по заявкам и кандидатам на напоминание;
- аккуратное разделение клиентского и внутреннего сценария.
- классификацию типа потока: продажи / поддержка / бронирование / партнерский портал / операции.

## Для каких задач подходит

- сервисные Telegram-боты;
- внутренние инструменты для команды;
- заявки и статусы;
- уведомления и напоминания;
- быстрые MVP под коммуникационные процессы.
