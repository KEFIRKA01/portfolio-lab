# Стол первичного HR-скрининга

`Стол первичного HR-скрининга` — большой витринный Telegram-бот.

## Фокус

- Нужно быстро собирать кандидатов, проводить базовый скрининг, координировать интервью и держать единый контур статусов по воронке найма.
- Показывает сложный HR-контур, где Telegram становится рабочим инструментом команды.
- Направление: `подбор, первичный скрининг, сбор анкеты, планирование интервью`

## Крупные блоки функционала

- воронка найма
- скоринг соответствия
- слоты интервью
- опрос менеджера
- сигналы задержек

## Ключевые сценарии

- Новый кандидат
- Первичный скрининг
- Назначение интервью
- Сбор фидбека
- Риск отказа

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `воронка найма, интервью, оценка кандидата`;
- хорошо продаёт нетипичные запросы заказчиков.
