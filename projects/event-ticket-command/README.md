# Командный центр продажи билетов

`Командный центр продажи билетов` — большой витринный Telegram-бот.

## Фокус

- Нужно продавать билеты прямо в Telegram, удерживать места, быстро добирать очередь ожидания и контролировать доступ на площадке.
- Показывает продуктовый Telegram-контур: продажи, лимиты, очередь, доступ и работу команды события.
- Направление: `билеты, удержание мест, очередь ожидания, QR-доступ`

## Крупные блоки функционала

- удержание мест
- очередь ожидания
- пакеты и апсейлы
- статусы билетов
- сценарий проверки доступа

## Ключевые сценарии

- Покупка билета
- Повышение до VIP
- Очередь ожидания
- Групповой заказ
- Проверка доступа

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `продажа билетов, удержание мест, очередь ожидания, контроль доступа`;
- хорошо продаёт нетипичные запросы заказчиков.
