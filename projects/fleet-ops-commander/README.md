# Командир транспортных операций

`Командир транспортных операций` — большой витринный Telegram-бот.

## Фокус

- Нужно координировать курьеров и водителей в Telegram, быстро обрабатывать инциденты и не терять SLA по доставкам и сервису.
- Расширяет портфель в сторону логистики и полевых операций, показывает зрелое мышление в очередях, эскалациях и ремонтах.
- Направление: `транспорт, диспетчеризация, инциденты, SLA, обслуживание`

## Крупные блоки функционала

- рейсы и SLA
- эскалация инцидентов
- переназначение
- окна сервиса
- дашборд диспетчера

## Ключевые сценарии

- Новый рейс
- Поломка на маршруте
- Переназначение курьера
- Сервисное окно
- VIP доставка

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `диспетчеризация, инциденты на маршруте, SLA, сервисные окна`;
- хорошо продаёт нетипичные запросы заказчиков.
