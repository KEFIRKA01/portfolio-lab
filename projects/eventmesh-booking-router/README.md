# Маршрутизатор бронирований EventMesh

## Витрина

Скриншоты и GIF складываются в `assets/`.

- shot-list: `SHOTLIST.md`
- assets: `assets/README.md`

Демонстрационный сервис маршрутизации заявок и бронирований между провайдерами, слотами и каналами входа.

![Swagger-витрина API](./assets/eventmesh-booking-router-02-main.png)

## Что показывает проект

- маршрутизацию между несколькими провайдерами;
- проверку слотов и защиту от конфликтов;
- резервную маршрутизацию;
- работу с несколькими каналами входа;
- сценарии срочных / VIP / внерабочих бронирований.

## Для каких задач подходит

- бронирование и планирование;
- API-интеграции с внешними провайдерами;
- Telegram/сайт/партнёрский канал как единый вход;
- B2B сервисы с маршрутизацией по слотам и командам.
