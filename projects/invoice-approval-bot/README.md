# Бот согласования счетов

`Бот согласования счетов` — большой витринный Telegram-бот.

## Фокус

- Нужно ускорить согласование счетов и расходов без потери контроля по лимитам, ролям, бюджетам и исключениям.
- Даёт сильный кейс в зоне внутренних бизнес-процессов, согласований, аудита и операционной автоматизации.
- Направление: `согласование расходов, маршрут счета, бюджетные лимиты`

## Крупные блоки функционала

- маршруты согласования
- бюджетные лимиты
- исключения
- аудит следов
- финансовые метрики

## Ключевые сценарии

- Новый счет
- Внеплановый расход
- Перерасход бюджета
- Срочное согласование
- Эскалация CFO

## Роли

- инициатор
- контролер
- согласующий
- владелец

## Категории

- операционные расходы
- маркетинг
- софт
- командировки
- срочно

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `маршрут согласования, контроль бюджета, финансовая автоматизация`;
- хорошо продаёт нетипичные запросы заказчиков.
