# Консьерж просмотров недвижимости

`Консьерж просмотров недвижимости` — большой витринный Telegram-бот.

## Фокус

- Нужно быстро квалифицировать интерес клиента к недвижимости, собрать базовые параметры сделки и не терять лиды на этапе записи на просмотр.
- Раскрывает бизнес-логику лидогенерации, квалификации, подборов и маршрутов просмотров.
- Направление: `недвижимость, квалификация лида, подборка объектов, просмотр`

## Крупные блоки функционала

- квалификация бюджета
- подборка объектов
- слоты просмотров
- маршрут агента
- ипотечный сценарий

## Ключевые сценарии

- Подбор аренды
- Покупка квартиры
- Коммерческое помещение
- Ипотечный пречек
- Маршрут на несколько просмотров

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `квалификация лида, маршрутизация объектов, план просмотров, ипотечный пречек`;
- хорошо продаёт нетипичные запросы заказчиков.
