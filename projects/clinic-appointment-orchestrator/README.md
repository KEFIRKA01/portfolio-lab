# Оркестратор записей клиники

`Оркестратор записей клиники` — большой витринный Telegram-бот.

## Фокус

- Нужно сократить хаос в записи на приём, собирать первичную информацию до визита и автоматически напоминать пациентам о шагах.
- Доказывает, что профиль умеет делать сложные сервисные боты с расписанием, формами, SLA и логикой статусов.
- Направление: `медицинская запись, первичный опрос, расписание врачей, повторные визиты`

## Крупные блоки функционала

- первичный опрос
- слоты и очереди
- напоминания
- маршрут пациента
- эскалация администратора

## Ключевые сценарии

- Первичная запись
- Повторный визит
- Онлайн-консультация
- Срочный слот
- Перенос записи

## Роли

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `запись, первичный опрос, уведомления, сервисная координация`;
- хорошо продаёт нетипичные запросы заказчиков.
