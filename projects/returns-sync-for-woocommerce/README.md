# Синхронизация возвратов для WooCommerce

Демонстрационный WooCommerce-плагин для возвратов, RMA-очереди и синхронизации событий возврата средств во внешние системы.

## Что показывает проект

- процессы e-commerce после покупки;
- возвраты и синхронизацию возврата средств;
- очередь RMA;
- повторные попытки доставки;
- брендированные и B2B-профили синхронизации.
