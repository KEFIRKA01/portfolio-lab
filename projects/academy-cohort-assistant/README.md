# Ассистент учебного потока

`Ассистент учебного потока` — большой витринный Telegram-бот.

## Фокус

- Нужно сопровождать учебные потоки в Telegram, напоминать о дедлайнах, собирать сигналы риска и не терять студентов в середине программы.
- Доказывает умение проектировать долгие рабочие сценарии с напоминаниями, обучающей логикой и вовлечением.
- Направление: `образовательный поток, онбординг группы, дедлайны, часы с ментором`

## Крупные блоки функционала

- группы и когорты
- сигналы вовлеченности
- дедлайны
- часы с ментором
- дашборд координатора

## Ключевые сценарии

- Онбординг
- Напоминание о дедлайне
- Запись на часы с ментором
- Риск отсева
- Запрос сертификата

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `образовательный сценарий, напоминания, координация потока, удержание`;
- хорошо продаёт нетипичные запросы заказчиков.
