# FlowDesk CRM Hub

## Витрина

Скриншоты и GIF складываются в `assets/`.

- shot-list: `SHOTLIST.md`
- assets: `assets/README.md`

Публичный демонстрационный проект, который показывает, как собрать прикладной контур приёма заявок: формы, webhook, CRM, Telegram-уведомления, журнал событий и платёжный колбэк.

## Что показывает проект

- приём лидов из разных каналов;
- нормализацию данных;
- антидубль на уровне сигнатуры заявки;
- оценку лидов и приоритизацию очереди;
- предпросмотр маршрутизации для корпоративных и брендированных сценариев;
- SLA-risk по горячим лидам;
- журнал событий;
- webhook-контур для внешних систем;
- связку заявки, статуса, заметок и платёжного колбэка.

## Для каких задач подходит

- CRM/API интеграции;
- формы заявок и post-submit логика;
- операторская панель;
- Telegram/email уведомления;
- сценарии платёжных колбэков;
- backend для сервисных сайтов и web-продуктов.

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Демо-сценарий

1. Отправка лида из формы или landing page.
2. Нормализация и антидубль.
3. Создание записи в операторском потоке.
4. Отправка webhook/уведомления.
5. Изменение статуса.
6. Комментарий по заявке, перерасчёт тегов и фиксация истории событий.
